# sh_contracthabitat

Paquete Node/TypeScript de contratos compartidos para Habitat. Actualmente concentra contratos de `CH09-AEM`.

## Camino rápido

```bash
npm install
npm run tsc
```

Prueba local declarada:

```bash
npm test
```
